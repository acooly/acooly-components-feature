Acooly安全组件 acooly-module-security

简介

基于apache-shiro封装的基于角色的安全管理框架，同时作为acooly框架的页面框架基础版本，其中系统的部分共也可以作为该框架的showcase使用。主要功能包括：

  • 资源，菜单，功能权限树形结构管理
  • 角色管理及资源权限分配，支持简单分级授权。
  • 支持基于URL及ant-pattern权限控制
  • 支持基于功能和按钮级的细粒度权限控制
  • 用户账户管理，密码加盐加密存储
  • 用户账户有效期，登录错误次数控制（锁定一段时间）
  • 支持用户密码规则配置定制（正则表单式）
  • 支持基于组织机构的外部扩展
  • 支持用户状态和类型的配置化扩展
  • 提供XSS和CSRF常规防御能力支持
  • 自动提供基于用户的权限，资源信息缓存（SCOPE=USER-RUNTIME）
  • 提供常规安全加密算法和摘要算法封装
  • 提供基于jcaptcha的随机验证码封装
  • 前端提供一套完成的基于easyUI的acooly封装前端管控后台框架
  • 前端提供基本操作（CRUD等）的封装JS库
  • 前端提供常用类型转换显示JS库扩展
  • 前端提供KingEditor多媒体编辑器封装（包括文件上传支持）
  • 前端提供首页portalet桌面功能
